# Rivals Hub Key System

This repository contains the **Rivals Hub Key Generator Website**.

## Live Website
[Click here to open the styled key website](https://officialmodded2.github.io/RivalsHubKeyWebsite/)

## How It Works
1. Complete the default tasks (YouTube, Page, Discord) **or** enter your own custom URL.
2. Click **Generate Key** after completing tasks.
3. Copy the key and use it in your Roblox hub loader.
4. Keys expire after a set time and cannot be reused.

## Files
- `index.html` → The website frontend
- `style.css` → Styling for the website
- `server.js` → Node.js backend for key generation and verification
